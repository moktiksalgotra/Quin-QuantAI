Quin-QuantAI API

Welcome to the Quin-QuantAI API. This API provides data analysis capabilities using AI.

Available Endpoints:

Health Check

GET /api/health

Check if the API is running.

Upload Dataset

POST /api/upload

Upload a CSV or Excel file for analysis.

Get Current Dataset

GET /api/dataset

Get information about the currently loaded dataset.

Chat

POST /api/chat

Ask questions about your data in natural language.

Reset Database

POST /api/reset

Reset the database.

For more information, please refer to the API documentation.
